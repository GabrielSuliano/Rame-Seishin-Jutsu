# Rame Seishin Jutsu

Site estatico em HTML, CSS e JavaScript, preparado para deploy na Vercel.

## Build

```bash
npm run build
```

O comando gera a pasta `dist/`, que e a saida usada pela Vercel.

## Configuracao na Vercel

- Framework Preset: `Other`
- Build Command: `npm run build`
- Output Directory: `dist`
